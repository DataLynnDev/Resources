Table 1: Historical Traffic Dataset (HTD)¶

Column Header Description
Column Name Description
traffic_id Unique identifier for each traffic data point.
location_id Identifier for each specific location (interconnected with RAED).
timestamp The time at which the traffic data was recorded (YYYY-MM-DD HH:MM:SS).
speed Average speed of vehicles (km/h) at the given time.
volume Number of vehicles passing through the location at the given time.
congestion Congestion level(continuous variable).

Table 2: Road Attribute and Event Dataset (RAED)¶

Column Header Description
location_id Identifier for each specific location (interconnected with HTD).
road_type Type of road - 'Highway', 'Main Road', 'Side Street', etc.
num_lanes Number of lanes on the road.
event_id Unique identifier for a specific event (like a concert, football match, etc.). Null if no event.
event_type Type of event if exists - 'Sporting', 'Concert', 'Festival', etc. Null if no event.
event_timestamp Start time of the event (YYYY-MM-DD HH:MM:SS). Null if no event.
weather Weather condition at the time - 'Sunny', 'Rainy', 'Snowy', etc.

# Load datasets
# htd = pd.read_csv('htd.csv', parse_dates=['timestamp'])
# raed = pd.read_csv('raed.csv', parse_dates=['event_timestamp'])

# Convert timestamps to the same granularity (hourly in this example)
htd['timestamp'] = pd.to_datetime(htd['timestamp'])
htd['timestamp'] = htd['timestamp'].dt.floor('H')

raed['event_timestamp'] = pd.to_datetime(raed['event_timestamp'])
raed['event_timestamp'] = raed['event_timestamp'].dt.floor('H')

# Merge datasets on both location_id and timestamp
merged_df = pd.merge(htd, raed, left_on=['location_id', 'timestamp'], right_on=['location_id', 'event_timestamp'], how='left')

# Fill missing values for event_type with 'No Event'
merged_df['event_type'] = merged_df['event_type'].fillna('No Event')

# Handling outliers
# This is just an example and you might need to adjust this based on your specific use case
Q1 = merged_df['volume'].quantile(0.25)
Q3 = merged_df['volume'].quantile(0.75)
IQR = Q3 - Q1

filter = (merged_df['volume'] >= Q1 - 1.5 * IQR) & (merged_df['volume'] <= Q3 + 1.5 *IQR)
merged_df = merged_df[filter]

# Mapping weather conditions to numerical values
weather_conditions_mapping = {'Sunny': 0, 'Rainy': 1, 'Snowy': 2}
merged_df['weather'] = merged_df['weather'].map(weather_conditions_mapping)

# Extract day of week, month and hour from timestamp
merged_df['day_of_week'] = merged_df['timestamp'].dt.dayofweek
merged_df['month'] = merged_df['timestamp'].dt.month
merged_df['hour'] = merged_df['timestamp'].dt.hour

# Create a binary feature indicating if there is an event or not
merged_df['has_event'] = merged_df['event_type'].apply(lambda x: 0 if x == 'No Event' else 1)

from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

# Selecting the independent variables (features) and the dependent variable (target)
features = htd[['traffic_id', 'speed', 'Sunny', 'Rainy', 'Snowy', 'hour', 'day', 'month', 'year']]
target = htd['congestion']

# Split the data into training set and test set
X_train, X_test, y_train, y_test = train_test_split(features, target, test_size=0.2, random_state=42)

# Create a Linear Regression object
lr = LinearRegression()

# Train the model
lr.fit(X_train, y_train)

# Make predictions using the testing set
y_pred = lr.predict(X_test)

# Print the coefficients of the model
print('Coefficients: \n', lr.coef_)

# Print the mean squared error
print('Mean squared error: %.2f' % mean_squared_error(y_test, y_pred))

# Print the coefficient of determination: 1 is perfect prediction
print('Coefficient of determination: %.2f' % r2_score(y_test, y_pred))
