Question 1¶

(Data Manipulation & Cleaning)
Given the IoT_data table, which is laden with missing values and potential anomalies:

  • Spot and handle any potential outliers in the product_quality score.

Solution:¶

Hints:¶

  1. Use Pandas for Data Manipulation

Pandas is a powerful Python library for data manipulation and analysis. Use it to load your dataset into a DataFrame for easier handling of missing values and outliers. Functions like pd.read_csv() and DataFrame.fillna() will be very handy.

  1. Identify Missing Values and Outliers through Statistical Methods

For missing values, you can use the DataFrame.isna().sum() function to count the number of missing entries for each column. For identifying outliers, consider statistical methods like Z-score or the Inter-Quartile Range (IQR). Pandas' quantile() function can help you calculate Q1 and Q3, essential values for the IQR method.

  1. Handle Missing Values and Outliers Appropriately

Once missing values and outliers are identified, decide on the most appropriate handling method based on the nature of your dataset. For missing values, you might choose to remove those entries, or you could impute them with the mean, median, or mode. For outliers, options could include removing them or capping them to a specific range. Use Pandas' DataFrame manipulation methods, like fillna() or where(), to implement these solutions.

# Impute missing values with the mean of the column
mean_vibration = df['machine_vibration'].mean()
df['machine_vibration'].fillna(mean_vibration, inplace=True)

# Step 3: Spot and Handle Outliers in 'product_quality'

# Identify outliers using IQR method

# Compute the first quartile (Q1) and third quartile (Q3) of the column
Q1 = df['product_quality'].quantile(0.25)
Q3 = df['product_quality'].quantile(0.75)

# Compute the Inter-Quartile Range (IQR)
IQR = Q3 - Q1

# Define bounds for the outliers
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

# Identify the outliers
outliers = df[(df['product_quality'] < lower_bound) | (df['product_quality'] > upper_bound)]

# Print the number of identified outliers
print(f"Identified {len(outliers)} outliers in product_quality.")

# Handle outliers by capping them to the IQR bounds
df['product_quality'] = np.where(df['product_quality'] < lower_bound, lower_bound, df['product_quality'])
df['product_quality'] = np.where(df['product_quality'] > upper_bound, upper_bound, df['product_quality'])

From the IoT_data, using data from devices located at 'location_code_X':

  • Calculate the probability that a product is defective.
  • Based on machine_temperature, can you infer if there's a significant difference in temperature when a product turns out to be defective?

Solution:¶

Hints:¶

  1. Data Filtering: Start by filtering the dataset to only include records where the device_location is 'location_code_X'.

  2. Calculate Defect Probability: Use the filtered data to calculate the probability of a product being defective. The probability can be calculated as the ratio of defective products to the total number of products.

  3. Temperature Significance Testing: Use a t-test to compare the machine_temperature between defective and non-defective products in the filtered dataset. This will help you assess if there's a significant difference in temperature when a product turns out to be defective.

# Import necessary libraries
import pandas as pd
from scipy import stats

# Step 1: Filter data for 'location_code_X'. We take location 1 as an example
filtered_data = df[df['device_location'] == 'location_code_1']

# Step 2: Calculate the probability that a product is defective in 'location_code_X'
total_products = len(filtered_data)
defective_products = len(filtered_data[filtered_data['defective'] == 1])

probability_defective = defective_products / total_products
print(f"The probability that a product is defective at location_code_X is: {probability_defective}")

# Step 3: Analyze if there's a significant difference in machine_temperature when a product is defective
# We'll use a t-test for this purpose

# Gather temperatures for defective and non-defective products
temp_defective = filtered_data[filtered_data['defective'] == 1]['machine_temperature']
temp_non_defective = filtered_data[filtered_data['defective'] == 0]['machine_temperature']

# Perform t-test
t_stat, p_value = stats.ttest_ind(temp_defective, temp_non_defective, equal_var=False)

# Check if the p_value is less than 0.05 for 95% confidence level
if p_value < 0.05:
    print("There is a significant difference in machine temperature when a product is defective.")
else:
    print("There is no significant difference in machine temperature when a product is defective.")
