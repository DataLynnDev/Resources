Q1¶

(Statistical methods and models, Data Analysis, Modeling and Projects) Using the CustomerInfo table, perform an exploratory data analysis to understand the characteristics of customers who have churned (Exited = 1) versus those who have not. What insights can you draw from this analysis?

Solution:

In [ ]:
import pandas as pd
import matplotlib.pyplot as plt

# Assuming that 'CustomerInfo' is our DataFrame

# Get the subset of customers who have churned
churned = CustomerInfo[CustomerInfo['Exited'] == 1]

# Get the subset of customers who have not churned
not_churned = CustomerInfo[CustomerInfo['Exited'] == 0]

# For each column, plot a histogram for churned and not churned customers
for column in CustomerInfo.columns:
    if column not in ['CustomerID', 'Exited']:
        plt.figure(figsize=(5, 2))

        plt.subplot(1, 2, 1)
        plt.hist(churned[column], bins=30, alpha=0.5, color='r', label='churned')
        plt.title(f'{column} (churned)')

        plt.subplot(1, 2, 2)
        plt.hist(not_churned[column], bins=30, alpha=0.5, color='b', label='not churned')
        plt.title(f'{column} (not churned)')

        plt.show()

Q2¶

(Machine learning, Theoretical Questions and Definitions) Based on your exploratory data analysis, build a logistic regression model to predict customer churn. What are the key features influencing customer churn according to your model? How would you validate the performance of this model?

Solution:

In [ ]:
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import classification_report
from sklearn.preprocessing import StandardScaler


# Assuming that 'CustomerInfo' is our DataFrame

# Select features and target variable
features = CustomerInfo.drop(['CustomerID', 'Exited'], axis=1)
target = CustomerInfo['Exited']

# Convert categorical features into dummy variables
features = pd.get_dummies(features)

# Split the data into training and test sets
X_train, X_test, y_train, y_test = train_test_split(features, target, test_size=0.2, random_state=42)

# Apply scaling to the training data
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)

# Apply the same scaling to the test data
X_test = scaler.transform(X_test)

# Create and train the logistic regression model
model = LogisticRegression()
model.fit(X_train, y_train)

# Make predictions on the test set
y_pred = model.predict(X_test)

# Print the classification report
print(classification_report(y_test, y_pred))

# Print the coefficients of the logistic regression model
coefficients = pd.DataFrame({'Feature': features.columns, 'Importance': model.coef_[0]})
print(coefficients.sort_values(by='Importance', key=abs, ascending=False))

Q3¶

(Programming and Databases, ML Algorithms and Techniques) Using the TransactionHistory table, create a new feature in the CustomerInfo table that represents the average transaction amount for each customer over the past 6 months. How does this new feature influence your logistic regression model? Solution:

In [ ]:
from datetime import datetime, timedelta
from sklearn.preprocessing import StandardScaler


# Assuming that 'CustomerInfo' and 'TransactionHistory' are our DataFrames

# Convert TransactionDate to datetime
TransactionHistory['TransactionDate'] = pd.to_datetime(TransactionHistory['TransactionDate'])

# Get the date 6 months ago from the latest transaction
six_months_ago = TransactionHistory['TransactionDate'].max() - timedelta(days=180)

# Filter transactions in the past 6 months
recent_transactions = TransactionHistory[TransactionHistory['TransactionDate'] > six_months_ago]

# Calculate the average transaction amount for each customer
average_transaction_amount = pd.DataFrame()
average_transaction_amount['average_TransactionAmount'] = recent_transactions.groupby('CustomerID')['TransactionAmount'].mean()

# Add this new feature to the CustomerInfo table
CustomerInfo_add = CustomerInfo.join(average_transaction_amount, on='CustomerID')

# Now, we can retrain our logistic regression model with the new feature

# Select features and target variable
features = CustomerInfo_add.drop(['CustomerID', 'Exited'], axis=1)
target = CustomerInfo_add['Exited']

# Convert categorical features into dummy variables
features = pd.get_dummies(features)

# Handle any missing values introduced by the join operation
features = features.fillna(0)

# Split the data into training and test sets
X_train, X_test, y_train, y_test = train_test_split(features, target, test_size=0.2, random_state=42)

# Apply scaling to the training data
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)

# Apply the same scaling to the test data
X_test = scaler.transform(X_test)

# Create and train the logistic regression model
model = LogisticRegression()
model.fit(X_train, y_train)

# Make predictions on the test set
y_pred = model.predict(X_test)

# Print the classification report
print(classification_report(y_test, y_pred))

# Print the coefficients of the logistic regression model
coefficients = pd.DataFrame({'Feature': features.columns, 'Importance': model.coef_[0]})
print(coefficients.sort_values(by='Importance', key=abs, ascending=False))
